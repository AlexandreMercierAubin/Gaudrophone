Le Gaudrophone est un concepteur d'instrument virtuel.
Grâce à ce logiciel, il devient possible de créer numériquement son propre instrument.
______________________________________________________________________________________

Modes d'éditions

Initialement, le Gaudrophone démarre en mode Ajouter touche.
Comme le nom du mode le dit, il est possible d'ajouter des touches en cliquant. 
La touche sera ajoutée à l'emplacement pointé par la souris lors du relâchement du clic.

Le mode Éditer touche permet de sélectionner une touche afin de modifié ses
charactéristiques, comme le son produit, ses bordures, sa dimension, sa forme, etc.
Il est aussi possible de déplacer une touche en la cliquant sur une touche, conservant 
le clic tout en la glissant et en relâchant le clic au nouvel emplacement désiré.

Le mode jouer permet de jouer de votre instrument en cliquant sur les touches ajoutées
et configurées à votre goût.
______________________________________________________________________________________

Menu d'édition

Dans le menu d'édition situé à la droite, il est possible de modifier les paramètres
de l'instrument.

Ces paramètres incluent entre autre :
-Le nom
-Le timbre
-Le metronome


Un onglet touche est situé dans le haut à gauche du menu d'édition.
Il est possible d'accèder à ce menu lorsqu'une touche est sélectionnée.
Une fois dans le menu, il est possible d'éditer les paramètres de la touche 
sélectionnée. 

Ces paramètres incluent entre autre :
-La forme
-La couleur ou l'image de fond
-La dimension
-Les bordures
-Le type (son ou fichier audio).
______________________________________________________________________________________

Sauvegarder un instrument

Il est possible de sauvegarder un instrument en cliquant sur le menu fichier suivit 
par le bouton enregistrer ou enregistrer sous.

La différence entre enregistrer et enregistrer sous est qu'enregistrer sous permets de choisir
l'emplacement où le fichier est enregistré, tandis que enregistrer va enregistrer à la dernière
place où cet instrument a été enregistré.

Dans l'éventualité où l'instrument n'a jamais été enregistré encore, enregistrer agit comme
enregistrer sous.
______________________________________________________________________________________

Importer un instrument

Il est possible d'importer un instrument sauvegardé à l'aide du menu importer se situant
dans l'onglet ficher.

Il suffit de choisir l'instrument à importer à l'aide de l'explorateur de fichiers.
